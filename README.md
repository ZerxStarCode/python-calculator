# Python Calculator (Tkinter)

A simple desktop calculator built with **Python** and **Tkinter**, packaged as a Windows executable using **PyInstaller**.

This project is a personal practice application focused on:
- Building graphical user interfaces with Tkinter
- Structuring a small Python application
- Packaging Python programs into standalone `.exe` files

---

## Features

- Basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Graphical interface using Tkinter
- Standalone executable for Windows
- Lightweight and easy to use
